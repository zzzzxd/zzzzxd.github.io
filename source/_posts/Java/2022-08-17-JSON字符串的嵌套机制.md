### 转义机制的嵌套：

转义机制被任意嵌套、组合，只要识别方的处理逻辑是正确的，就不会发生命令识别错误或数据识别错误。转义机制是一个安全性完备的机制。

每增加一层转义机制，就需要增加一层还原机制。

### **JSON字符串的多层嵌套**

```java
<dependency>
    <groupId>com.alibaba</groupId>
    <artifactId>fastjson</artifactId>
    <version>1.2.76</version>
</dependency>

/**
 * 多重嵌套转义测试
 */
@Test
public void multipleEscape() {
    JSONObject jsonObject = new JSONObject();
    String s1 = "hello world.";
    for (int i = 0; i < 4; i++) {
        jsonObject.set("test", s1);
        s1 = jsonObject.toString();
        System.out.println(s1);
    }
}

输出：
{"test":"hello world."}
{"test":"{\"test\":\"hello world.\"}"}
{"test":"{\"test\":\"{\\\"test\\\":\\\"hello world.\\\"}\"}"}
{"test":"{\"test\":\"{\\\"test\\\":\\\"{\\\\\\\"test\\\\\\\":\\\\\\\"hello world.\\\\\\\"}\\\"}\"}"}
```
