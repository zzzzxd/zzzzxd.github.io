**在idea 初始化本地项目后**

1. 在github创建仓库

2. 添加git远程仓库绑定:git remote add origin https://github.com/zzzzxd/xxxx 当前没有追踪远程分支,拉取时候绑定远程分支:git pull origin master 

**常见的其他操作**

- - 删除项目当前分支对远程分支的关联

 ```shell
 git remote rm origin
 ```

- - 项目以http方式重新关联

```shell
 git remote add origin http://gitlab.xxxxxxxx.cn:xxxx/xxxxxx/mall.git
```

- - 项目中本地分支和远程分支关联

```shell
git pull git branch --set-upstream-to=origin/de
```
