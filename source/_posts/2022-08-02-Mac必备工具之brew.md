>brew 是 Mac 下的一个包管理工具，类似于 centos 下的 yum，可以很方便地进行安装/卸载/更新各种软件包，例如：nodejs, elasticsearch, kibana, mysql, mongodb 等等，可以用来快速搭建各种本地环境，程序员必备工具

### 安装

可以使用如下命令进行安装:

```shell
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```

### 基本使用

安装/卸载/更新

```shell
brew install nodejs
brew upgrade nodejs
brew remove  nodejs
```

其它命令

```shell
brew list                   # 列出当前安装的软件
brew search nodejs          # 查询与 nodejs 相关的可用软件
brew info nodejs            # 查询 nodejs 的安装信息
```

Services 命令

 brew services 是一个非常强大的工具，可以用来管理各种服务的启停，有点像 linux 里面的 services

```shell
brew services list    //列出当前管理的服务
```

启动/停止/重启

```shell
brew start   nginx
brew stop    nginx
brew restart nginx
```

### 常见路径

- 配置路径：`/usr/local/etc/`
- 日志路径：`/usr/local/var/log`
- 安装路径   `/usr/local/Cellar`
