## 前言

有关对象的创建、对象在内存中的布局，今天我们通过几道面试题来梳理一下。

## 请解释一下对象的创建过程？

1. class loading
2. class linking (verification, preparation, resolution) - 静态变量赋默认值
3. class initializing- 静态变量赋初始值
4. 申请对象内存
5. 成员变量赋默认值
6. 调用构造方法<init>
   1. 成员变量顺序赋初始值
   2. 执行构造方法语句

## 对象在内存中的存储布局？

### 普通对象

1. 对象头：markword  8
2. ClassPointer指针：-XX:+UseCompressedClassPointers 为4字节 不开启为8字节
3. 实例数据
   1. 引用类型：-XX:+UseCompressedOops 为4字节 不开启为8字节 
      Oops Ordinary Object Pointers
4. Padding对齐，8的倍数

### 数组对象

1. 对象头：markword 8
2. ClassPointer指针同上
3. 数组长度：4字节
4. 数组数据
5. 对齐 8的倍数
