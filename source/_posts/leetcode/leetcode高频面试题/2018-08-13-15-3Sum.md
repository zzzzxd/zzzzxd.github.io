### 题目链接

​	https://leetcode.com/problems/3sum/

### 题目描述

Given an integer array nums, return all the triplets `[nums[i], nums[j], nums[k]]` such that `i != j`, `i != k`, and `j != k`, and `nums[i] + nums[j] + nums[k] == 0`.

Notice that the solution set must not contain duplicate triplets.

 

**Example 1:**

```
Input: nums = [-1,0,1,2,-1,-4]
Output: [[-1,-1,2],[-1,0,1]]
Explanation: 
nums[0] + nums[1] + nums[1] = (-1) + 0 + 1 = 0.
nums[1] + nums[2] + nums[4] = 0 + 1 + (-1) = 0.
nums[0] + nums[3] + nums[4] = (-1) + 2 + (-1) = 0.
The distinct triplets are [-1,0,1] and [-1,-1,2].
Notice that the order of the output and the order of the triplets does not matter.
```

**Example 2:**

```
Input: nums = [0,1,1]
Output: []
Explanation: The only possible triplet does not sum up to 0.
```

**Example 3:**

```
Input: nums = [0,0,0]
Output: [[0,0,0]]
Explanation: The only possible triplet sums up to 0.
```

 

**Constraints:**

- `3 <= nums.length <= 3000`
- `-105 <= nums[i] <= 105`

### 题目解析

- 中等难度，不错的题目。先理解题目，要求输出的三元组 是元素的值，并且list里面的3元组不能重复
- 可以这样想3元组和为0，3个元素一定是小中大的关系，可以枚举小的元素，再去找中和大的元素。可以将元素排序，这样依次枚举，枚举的一定是小的元素，巧妙的避免重复。

### 代码

```java
class Solution {
  public static List<List<Integer>> threeSum(int[] nums) {
        final ArrayList<List<Integer>> list = new ArrayList<>();
        if(nums.length <= 2) return list;
        Arrays.sort(nums);
        for (int i = 0; i < nums.length; i++) {
            if (i > 0 && nums[i]==nums[i-1])continue;
            int p1 = i+1,p2 = nums.length-1,target = 0-nums[i];
            while (p1 < p2){
                if(nums[p1]+nums[p2] == target){
                    final ArrayList<Integer> tmp = new ArrayList<>();
                    tmp.add(nums[i]);
                    tmp.add(nums[p1++]);
                    tmp.add(nums[p2--]);
                    list.add(tmp);
                    while (p2 > p1 && nums[p2] == nums[p2+1]) p2--;
                    while (p1 < p2 && nums[p1] == nums[p1-1]) p1++;
                }else if(nums[p1]+nums[p2] < target){
                    p1++;
                }else
                    p2--;
            }
        }
        return list;
    }
}
```
