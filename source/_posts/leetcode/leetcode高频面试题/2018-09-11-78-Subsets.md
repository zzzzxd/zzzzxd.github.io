### 题目链接

​	https://leetcode.com/problems/subsets/

### 题目描述

​	Given an integer array `nums` of **unique** elements, return *all possible subsets (the power set)*.

The solution set **must not** contain duplicate subsets. Return the solution in **any order**.

 

**Example 1:**

```
Input: nums = [1,2,3]
Output: [[],[1],[2],[1,2],[3],[1,3],[2,3],[1,2,3]]
```

**Example 2:**

```
Input: nums = [0]
Output: [[],[0]]
```

 

**Constraints:**

- `1 <= nums.length <= 10`
- `-10 <= nums[i] <= 10`
- All the numbers of `nums` are **unique**.

​	

### 题目解析

- 求子集，从左到右递归枚举，加上回溯，收集起来即可。

### 代码

```java
class Solution {
   public static List<List<Integer>> subsets(int[] nums) {
        final List<List<Integer>> res = new ArrayList<>();
        final List<Integer> path = new ArrayList<>();
        final ArrayList<Integer> fin = new ArrayList<>();
        fin.add(nums[0]);
        for (int i = 1; i < nums.length; i++) {
            fin.add(nums[i]);
            process(res,path,nums,0,i);
        }
        res.add(new ArrayList<>());
        res.add(fin);
        return res;
    }
    public static void process(List<List<Integer>> res,List<Integer> path,int[] nums,int index,int lim){
        if (lim == 0){
            final ArrayList<Integer> list = new ArrayList<>(path);
            res.add(list);
            return;
        }
        if (index==nums.length || nums.length-index < lim){
            return;
        }
        path.add(nums[index]);
        process(res,path,nums,index+1,lim-1);
        path.remove(path.size()-1);
        process(res,path,nums,index+1,lim);
    }
}
```
