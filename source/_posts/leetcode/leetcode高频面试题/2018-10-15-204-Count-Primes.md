### 题目描述

​	Given an integer `n`, return *the number of prime numbers that are strictly less than* `n`.

 

**Example 1:**

```
Input: n = 10
Output: 4
Explanation: There are 4 prime numbers less than 10, they are 2, 3, 5, 7.
```

**Example 2:**

```
Input: n = 0
Output: 0
```

**Example 3:**

```
Input: n = 1
Output: 0
```

 

**Constraints:**

- `0 <= n <= 5 * 106`

​	

### 题目解析

- 数数素数，利用打表法，开数组，把素数打出来，再数就可以了。我们这里说的数素数指的是 埃及素数筛选法，时间复杂度是loglog(N)。

### 代码

```java
class Solution {
    final static TreeSet<Integer> set = new TreeSet<>();
    final static ArrayList<Integer> list = new ArrayList<>();
    public static int countPrimes(int n) {
        if(set.isEmpty()){
            boolean[] primer = new boolean[9000000]; //false代表是
            for(int i = 2;i < 9000000;i++){
                if(!primer[i]){
                    list.add(i);
                    set.add(i);
                    for (int j = i * 2;j < 9000000;j += i){
                        primer[j] = true;
                    }
                }
            }
        }
        final Integer primer_floor = set.floor(n);
        final int loc = list.indexOf(primer_floor);
        if (set.contains(n)){
            return loc;
        }else
            return loc +1;
    }
}
```
