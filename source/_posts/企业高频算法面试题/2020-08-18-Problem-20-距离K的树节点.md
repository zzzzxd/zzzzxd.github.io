### 题目

给定三个参数：
二叉树的头节点head，树上某个节点target，正数K
从target开始，可以向上走或者向下走，返回与target的距离是K的所有节点	

### 题目解析

- 可以往上走，那就是3个方向，找距离为K的，那不就是图的bfs吗。所以说思路要打开。树是不带父亲的路径的，需要自己用有序表存一下。

### 代码

```java
public static List<Node> getKDistanceNodes(Node root, Node target, int K){
        final HashMap<Node, Node> parent = new HashMap<>();
        final HashSet<Node> visited = new HashSet<>();
        dfs(root,null,parent);
        final ArrayList<Node> list = new ArrayList<>();
        final Queue<Node> queue = new LinkedList<Node>();
        int next_step = 0,next_batch = 1;

        queue.add(target);
        while (!queue.isEmpty()){
            int temp_count = 0,temp_next_batch = 0;
            while (temp_count < next_batch){
                final Node node = queue.poll();
                visited.add(node);
                if(next_step == K){
                    list.add(node);
                }
                if(node.left != null && !visited.contains(node.left)){
                    queue.add(node.left);
                    temp_next_batch++;
                }
                if(node.right != null && !visited.contains(node.right)){
                    queue.add(node.right);
                    temp_next_batch++;
                }
                if(parent.get(node) != null && !visited.contains(parent.get(node))){
                    queue.add(parent.get(node));
                    temp_next_batch++;
                }
                temp_count++;
            }
            if(next_step == K) break;
            next_step++;
            next_batch =temp_next_batch;

        }
        return list;

    }
    public static void dfs(Node cur,Node parent,Map<Node,Node> map){
        if(cur == null){
            return;
        }else{
            map.put(cur,parent);
            dfs(cur.left,cur,map);
            dfs(cur.right,cur,map);
        }
    }

```
