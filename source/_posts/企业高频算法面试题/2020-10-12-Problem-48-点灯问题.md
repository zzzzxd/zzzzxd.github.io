### 题目

给定一个数组arr，长度为N，arr中的值不是0就是1。arr[i]表示第i栈灯的状态，0代表灭灯，1代表亮灯
每一栈灯都有开关，但是按下i号灯的开关，会同时改变i-1、i、i+1栈灯的状态
问题一：如果N栈灯排成一条直线,请问最少按下多少次开关？
i为中间位置时，i号灯的开关能影响i-1、i和i+1
0号灯的开关只能影响0和1位置的灯
N-1号灯的开关只能影响N-2和N-1位置的灯

问题二：如果N栈灯排成一个圈,请问最少按下多少次开关,能让灯都亮起来
i为中间位置时，i号灯的开关能影响i-1、i和i+1
0号灯的开关能影响N-1、0和1位置的灯
N-1号灯的开关能影响N-2、N-1和0位置的灯

### 题目解析

- 经典的点灯问题。贪心的思路，尽可能少的点灯。然后 对于圈，无非是第一个位置比较特殊，单独对待一下。

### 代码

```java
//无环的点灯
    public static int method01(int[] arr){
        int N = arr.length;
        if(N == 0) return 0;
        if(N==1) return arr[0]^1;
        if(N==2) return arr[0]!=arr[1]?Integer.MAX_VALUE : arr[0]^1;
        final int[] arr2 = Arrays.copyOf(arr,  arr.length);
        //可能性一
        int p1 = 0;
        p1 = loop2(arr,p1);
        //可能性二
        int p2 = 1;
        arr2[0] ^= 1;
        arr2[1] ^=1;
        p2 = loop2(arr2,p2);
        return Math.min(p1,p2);
    }
 public static int loop2(int[] arr,int p){
        for (int i = 2; i < arr.length; i++) {
            if (arr[i - 1] == 0) { //灭灯
                p++;
                arr[i - 1] ^= 1;
                arr[i] ^= 1;
                arr[(i+1)%arr.length]^=1;
            }
        }
        if (arr[arr.length-1] == 0 || arr[0] == 0) p = Integer.MAX_VALUE;
        return p;
    }
 //循环点灯
    public static int method2(int[] arr){
        int N = arr.length;
        if(N == 0) return 0;
        if(N==1) return arr[0]^1;
        if(N==2) return arr[0]!=arr[1]?Integer.MAX_VALUE : arr[0]^1;
        int[] arr2 = Arrays.copyOf(arr,  arr.length);
        //前两位，你是不知道该取什么情况的，因为它不在我们说的普遍的位置，普遍的位置就希望左边过来的都是1
        //可能性一  0 不变，1不变
        int p1 = 0;
        p1 = loop2(arr2,p1);
        //可能性二 0不变 1变
        arr2 = Arrays.copyOf(arr,  arr.length);
        int p2 = 1;
        arr2[0] ^=1;
        arr2[1] ^=1;
        arr2[2] ^=1;
        p2 = loop2(arr2,p2);
        //可能性三 0变 1不变
        arr2 = Arrays.copyOf(arr,  arr.length);
        int p3 = 1;
        arr2[0] ^= 1;
        arr2[1] ^=1;
        arr2[N-1] ^=1;
        p3 = loop2(arr2,p3);
        //可能性四 0变 1变
        int p4 = 2;
        arr2 = Arrays.copyOf(arr,  arr.length);
        arr2[N-1] ^=1;
        arr2[2] ^=1;
        p4 = loop2(arr2,p4);
        return Math.min(Math.min(Math.min(p1,p2),p3),p4);
    }
```
