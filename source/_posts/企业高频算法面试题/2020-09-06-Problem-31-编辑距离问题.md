### 题目

​	一个字符串 可以有插入（insert）、删除（delete）、替换（replace）操作，每种操作都有对应的代价。求一个字符串变成另一个最小的代价。

### 题目解析

1. 经典的题目。动态规划中的行列模型。列举所有的可能性。

### 代码

```
 //方法：编辑距离问题，考察代价最小的最优解 ，典型的行列模型，关键在于，找准转换的所有可能性
    public static int MyminCost(String s1, String s2, int ic, int dc, int rc) {
        final char[] arr1 = s1.toCharArray();
        final char[] arr2 = s2.toCharArray();
        int M = arr1.length,N = arr2.length;
        final int[][] dp = new int[M+1][N+1];
        dp[0][0] = 0;
        for(int i = 1;i < M+1;i++){
            dp[i][0] = i*dc;
        }
        for(int j = 1;j < N+1;j++){
            dp[0][j] = j*ic;
        }
        for(int i = 1;i <= M;i++){
            for(int j = 1; j <= N;j++){
                dp[i][j] = dp[i-1][j] + dc;
                if(arr1[i-1]==arr2[j-1]) dp[i][j] = Math.min(dp[i][j],dp[i-1][j-1]);
                dp[i][j] = Math.min(dp[i][j],dp[i-1][j-1]+rc);
                dp[i][j] = Math.min(dp[i][j],dp[i-1][j-1]+ic+dc);
                dp[i][j] = Math.min(dp[i][j],dp[i][j-1]+ic);
            }
        }
        return dp[M][N];
    }
```
