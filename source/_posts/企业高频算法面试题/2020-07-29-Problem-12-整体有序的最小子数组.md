### 题目

​	给定一个数组arr，只能对arr中的一个子数组排序，但是想让arr整体都有序 返回满足这一设定的子数组中，最短的是多长。

leetcode 链接:https://leetcode.com/problems/shortest-unsorted-continuous-subarray/

### 题目解析

- 经典题目，简单分析一下，假设第一个元素比最后一个元素大，降序，那只能整体排序。如果第一个元素不是最小的，最后一个元素也不是最大的，他们都得参与排序，那么整体也得排序。
- 推广一下，也就是，对于当前元素 只有右面存在一个元素比我小就完蛋，当前元素就得参与排序，同理，左边有一个比我大的，就完蛋 我一样得参与排序，这就有了，我们分别从左边遍历，找到最后一个需要参与排序的元素， 从右面遍历找到参与排序的第一个元素，搞定！

### 代码

```java
 public static int getMinSubLen(int[] nums){
        if(nums.length == 0 || nums.length == 1) return 0;
        int max_l = nums[0],index_l = 0,min_r = nums[nums.length-1],index_r = nums.length-1;
        for (int i = 1; i < nums.length; i++) {
            if(nums[i] >= max_l ){
                max_l = nums[i];
            }else{
                index_l = i;
            }

            if(nums[nums.length-1-i] <= min_r){
                min_r = nums[nums.length-1-i];
            }else{
                index_r = nums.length-1-i;
            }
        }
        return index_l - index_r + 1 < 0 ? 0: index_l - index_r + 1;

    }
```
