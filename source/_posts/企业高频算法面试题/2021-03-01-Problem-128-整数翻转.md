### 题目

给你一个32位的有符号整数x
返回将x中的数字部分反转后的结果
反转后整数超过 32 位的有符号整数的范围就返回0
假设环境不允许存储 64 位整数（有符号或无符号）
Leetcode题目：https://leetcode.com/problems/reverse-integer/

### 题目解析

- 注意，给的就是整数,可以进行翻转溢出的提前判断

### 代码

```java
class Solution {
   public static int reverse(int x) {
        boolean neg = x < 0;
        x = neg ? x : -x;   //溢出是两个方向的，统一变负数处理，可以换成一个方向上 
        int m = Integer.MIN_VALUE / 10;
        int o = Integer.MIN_VALUE % 10;
        int res = 0;
        while (x != 0){
            if(res < m || res==m && x%10 < o)
                return 0;
            res = res * 10 + x%10;
            x /= 10;
        }
        return neg ? res : Math.abs(res);  //理论会有 负数没溢出，转绝对值溢出的情况，所以应该特殊处理，但这个题输入是一个整数，不会出现反转是-2147483647
    }
}
```
