### 题目

​	数组中所有数都异或起来的结果，叫做异或和 给定一个数组arr，可以任意切分成若干个不相交的子数组
其中一定存在一种最优方案，使得切出异或和为0的子数组最多 返回这个最多数量

### 题目解析

- 看到子数组先想DP ，打出前缀和，自己异或自己是0，利用这几点，题目就很简单了。

### 代码

```java
  //动态规划，利用异或的前缀和
    public static int MosteorZeroParts2(int arr[]){
        if(arr == null || arr.length == 0) return 0;
        int[] pre_xor = new int[arr.length],dp = new int[arr.length];
        final HashMap<Integer, Integer> map = new HashMap<>();
        pre_xor[0] = arr[0];
        map.put(0,-1); //初始值，防止前面整个数组异或和为0的情况 ,注意踩坑
        map.put(arr[0],0);
        dp[0] = arr[0] == 0? 1 : 0;
        for(int i = 1;i < arr.length;i++){
            pre_xor[i] =  pre_xor[i-1] ^ arr[i];
            dp[i] =  map.get(pre_xor[i]) == null ? dp[i-1] : Math.max((map.get(pre_xor[i])<0?0:dp[map.get(pre_xor[i])])+1,dp[i-1]);
            map.put(pre_xor[i],i);
        }
        return dp[arr.length-1];
    }
```
