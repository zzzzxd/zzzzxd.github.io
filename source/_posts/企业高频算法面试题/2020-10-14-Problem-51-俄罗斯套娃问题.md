### 题目

​	俄罗斯套娃问题

​	Leetcode 原题：

​	https://leetcode.com/problems/russian-doll-envelopes/

### 题目解析

- 经典题目，是LIS问题的应用。有长和高两个维度，我们将信封按照长升序，并且长相等时按照宽降序。这样做的目的就是，当我们把宽单独拿出来，此时的最长递增子序列的长度就是嵌套层次。而如果宽也是升序排序，此时的LIS长度 就会因为长度相等而产生干扰，而宽度降序就会直接排除掉干扰。

### 代码

```java
class Solution {
    private static class Model{
        int edge;
        int height;
        public Model(int edge, int height) {
            this.edge = edge;
            this.height = height;
        }
    }
    public static int maxEnvelopes(int[][] envelopes) {
        if(envelopes == null || envelopes.length == 0) return 0;
        Model[] model = new Model[envelopes.length];
        for (int i = 0; i < envelopes.length; i++) {
            model[i] = new Model(envelopes[i][0],envelopes[i][1]);
        }
        Arrays.sort(model,(t1,t2)->{
            if (t1.edge != t2.edge) return t1.edge - t2.edge;
            else
                return t2.height -t1.height;
        });
        final int[] lis = new int[envelopes.length];
        for (int i = 0; i < model.length; i++) {
            lis[i] = model[i].height;
        }
        return getLis(lis);
    }
    public static int getLis(int[] arr){
        int[] end = new int[arr.length];
        end[0] = arr[0];
        int valid_loc = 0;
        for (int i = 1;i < arr.length;i++){
            int l = 0,r = valid_loc;
            while (l < r){
                int mid = l + (r-l)/2;
                if(end[mid] >= arr[i]){
                    r = mid;
                }else
                    l = mid +1;
            }
            if(end[l] < arr[i]){
                end[l+1] = arr[i];
                valid_loc++;
            }else{
                end[l] = arr[i]; //更新对应长度的子序列的最小值
            }
        }
        return valid_loc+1;
    }
}
```
