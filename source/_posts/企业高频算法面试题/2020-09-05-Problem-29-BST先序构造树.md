### 题目

​		已知一棵搜索二叉树上没有重复值的节点，现在有一个数组arr，是这棵搜索二叉树先序遍历的结果
请根据arr生成整棵树并返回头节点

### 题目解析

- 就是递归在区间l-r中创建树,寻找左子树，右子树时候，因为是寻找第一个大于的,用二分正好，总体就是nlogn的

### 代码

```java
public static class TreeNode {
        int val;
        TreeNode left;
        TreeNode right;
        TreeNode() {}
        TreeNode(int val) { this.val = val; }
        TreeNode(int val, TreeNode left, TreeNode right) {
            this.val = val;
            this.left = left;
            this.right = right;
        }

        @Override
        public String toString() {
            return String.valueOf(val);
        }
    }
    //方法一：就是递归在区间l-r中创建树,寻找左子树，右子树时候，因为是寻找第一个大于的,用二分正好，总体就是nlogn的
    public  TreeNode bstFromPreorder(int[] preorder) {
        if( preorder == null || preorder.length == 0) return null;
        if(preorder.length == 1) return new TreeNode(preorder[0]);
        else
            return  process(preorder,0,preorder.length-1);
    }

    private TreeNode process(int[] preorder,int l,int r) {
        if(l > r) return null;
        if(l == r) return new TreeNode(preorder[l]);
        final TreeNode root = new TreeNode(preorder[l]);
        final int bigger_index = binarySearch(preorder, l + 1, r, preorder[l]);
        final TreeNode leftChild = process(preorder, l + 1, bigger_index - 1);
        final TreeNode rightChild = process(preorder, bigger_index, r);
        root.left = leftChild;root.right = rightChild;
        return root;
    }
        //查找第一个大于key的下标,-1为不存在
    private int binarySearch(int[] preorder,int from,int to,int key){
        int l = from,r = to;
        while (l < r){
            int mid = l + (r-l)/2;
            if(preorder[mid] > key){
                r = mid;
            }else{
                l = mid+1;
            }
        }
        return preorder[l] > key ? l : l+1;
    }
```
