### 题目

​	一个数组中只有两种字符'G'和’B’，可以让所有的G都放在左侧，所有的B都放在右侧 或者可以让所有的G都放在右侧，所有的B都放在左侧但是只能在相邻字符之间进行交换操作，返回至少需要交换几次

### 题目解析

- 这个题其实很简单，就是怕被说辞给吓住了，G在左侧，因为是最少的交换，所以一定从左侧开始让G移动。这样求的G在左侧的最少移动次数。同理，求的B在左侧的最少移动次数，也就是G在右侧，比较这两种情况

### 代码

```java
// 可以让G在左，或者在右
	public static int minSteps2(String s) {
		if (s == null || s.equals("")) {
			return 0;
		}
		char[] str = s.toCharArray();
		int step1 = 0;
		int step2 = 0;
		int gi = 0;
		int bi = 0;
		for (int i = 0; i < str.length; i++) {
			if (str[i] == 'G') { // 当前的G，去左边   方案1
				step1 += i - (gi++);
			} else {// 当前的B，去左边   方案2
				step2 += i - (bi++);
			}
		}
		return Math.min(step1, step2);
	}

```
