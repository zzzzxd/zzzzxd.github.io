### 题目

如果一个节点X，它左树结构和右树结构完全一样,那么我们说以X为头的树是相等树。给定一棵二叉树的头节点head ，返回head整棵树上有多少棵相等子树

### 题目解析

- 既然是比较树是不是一样的，那我们可以通过序列化的方式来描述一棵树。运用二叉树的递归套路，返回以head为头节点有多少个相等树，同时返回序列化结果。那么当拿到左右孩子的info信息时，便可以决定当前节点的相等子树个数。此外，比较序列化结果时，可以做一下hash。这样整体则是O(N)的运算。

### 代码

```java
public static class Info {
        public int ans;
        public String str;

        public Info(int a, String s) {
            ans = a;
            str = s;
        }
    }
//方法一:朴素的做法，遍历每个节点来求相等节点个数,判断是不是相等,
    public static int isSameTree(Node node){
        if(node == null) return 0;
        return process1(node);
    }

    private static int process1(Node node) {
        if(node == null) return 0;
        final int leftNum = process1(node.left);
        final int rightNum = process1(node.right);
        int res = leftNum + rightNum;
        final String strLeft = preOrderSquence(node.left);
        final String strRight = preOrderSquence(node.right);
        return strLeft.equals(strRight) ? res+1:res;
    }
    private static String preOrderSquence(Node node){
        if (node == null) return "null,";
        String str = node.value+",";
        final String leftStr = preOrderSquence(node.left);
        final String rightStr = preOrderSquence(node.right);
        return str + leftStr + rightStr;
    }
//改进: 对序列化串做hash。时间复杂度O(N)
    public static int sameNumber2(Node head) {
        String algorithm = "SHA-256";
        Hash hash = new Hash(algorithm);
        return process(head, hash).ans;
    }
    public static Info process(Node head, Hash hash) {
        if (head == null) {
            return new Info(0, hash.hashCode("#,"));
        }
        Info l = process(head.left, hash);
        Info r = process(head.right, hash);
        int ans = (l.str.equals(r.str) ? 1 : 0) + l.ans + r.ans;
        String str = hash.hashCode(String.valueOf(head.value) + "," + l.str + r.str);
        return new Info(ans, str);
    }
```
