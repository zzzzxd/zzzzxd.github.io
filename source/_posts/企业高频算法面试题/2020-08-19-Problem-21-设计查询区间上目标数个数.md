### 题目

数组为{3, 2, 2, 3, 1}，查询为(0, 3, 2)，意思是在数组里下标0~3这个范围上，有几个2？答案返回2。
假设给你一个数组arr，对这个数组的查询非常频繁，都给出来。

请返回 所有查询的结果

### 题目解析

- 数据结构的题目 通常都是考察某种思想，而且这种思想 在工作中，碰到的各种组建都会有应用到的地方，是可以落地使用的。
- 此题可以用一种反向索引表，也就是key是待查的元素，v是一个list，存的是递增的index。查找时可以在list上用二分查找。

### 代码

```java
public static class Finder{
        int[] num;
        HashMap<Integer, List<Integer>>  re_index;
        public  void build(int[] arr){
            num = Arrays.copyOfRange(arr,0,arr.length);
            re_index = new HashMap<>();
            for (int i = 0; i < num.length; i++) {
                if(re_index.containsKey(num[i]) ) {
                    re_index.get(num[i]).add(i);
                }else{
                    final ArrayList<Integer> list = new ArrayList<>();
                    list.add(i);
                    re_index.put(num[i],list);
                }
            }
        }
        public int find(int l,int r,int elem){
            final List<Integer> list = re_index.get(elem);
            if (list == null) return 0;
            final int a = binarySearch(list, l);
            final int b = binarySearch(list, r + 1);
            return b-a;
        }
        public static int binarySearch(List<Integer> list,int elem){
            int l = 0,r = list.size()-1;
            while (l < r){
                int mid = l + (r-l)/2;
                if(list.get(mid) >= elem)
                    r = mid;
                else
                    l = mid+1;
            }
            return list.get(l) >= elem ? l:l+1;
        }
    }
    public static void main(String[] args) {
            final Finder finder = new Finder();
            finder.build(new int[]{3, 2, 2, 3, 1});
            System.out.println(finder.find(0, 3, 1));
        }
```
