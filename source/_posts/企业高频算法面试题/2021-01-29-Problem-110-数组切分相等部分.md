### 题目

给定一个正数数组arr，长度一定大于6（>=7）
一定要选3个数字做分割点，从而分出4个部分，并且每部分都有数
分割点的数字直接删除，不属于任何4个部分中的任何一个。
返回有没有可能分出的4个部分累加和一样大
如：{3,2,3,7,4,4,3,1,1,6,7,1,5,2}
可以分成{3,2,3}、{4,4}、{1,1,6}、{1,5,2}。分割点是不算的！

### 题目解析

- 一定从最基础的思路开始。寻找可能性的题目都离不开枚举，关键看如何去枚举。

### 代码

```java
public static boolean canSplit(int[] arr){
        final HashSet<Integer> set = new HashSet<>();
        final HashMap<Integer, Integer> map = new HashMap<>();
        int sum = 0;
        for (int i = arr.length-1; i >= 0; i--) {
            sum+= arr[i];
            map.put(sum,i);
            set.add(sum);
        }
        int part = 0;
        for (int i = 1;i < arr.length-5;i++){  //枚举第一刀的位置 ,至少留5个元素 保证4部分
            part += arr[i-1];
            int cut2Sum = sum - (arr[i] + part * 2);
            if(set.contains(cut2Sum)){
                int cut3Sum = cut2Sum - arr[map.get(cut2Sum)] -  part;
                if (set.contains(cut3Sum)){
                    int lastPartSum = cut3Sum - arr[map.get(cut3Sum)];
                    if(lastPartSum == part)
                        return true;
                }
            }
        }
        return false;
    }
```
