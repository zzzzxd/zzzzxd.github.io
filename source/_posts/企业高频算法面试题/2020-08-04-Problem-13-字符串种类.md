### 题目

​	只由小写字母（a~z）组成的一批字符串 都放在字符类型的数组String[] arr中 如果其中某两个字符串所含有的字符种类完全一样 就将两个字符串算作一类 比如：baacbba和bac就算作一类 返回arr中有多少类？

### 题目解析

- 很容易想到，一个字符串用一个map表来表示只与种类有关。但实际上26个字母，26位 利用一个整数就可以表示。所以有几个整数就是有几类。

### 代码

```java
public static int countKind(String[] strings){
        final HashSet<Integer> set = new HashSet<>();
        for (int i = 0; i < strings.length; i++) {
            set.add(getInt(strings[i]));
        }
        return set.size();
    }

    private static Integer getInt(String string) {
        int res = 0;
        for (int i = 0; i < string.length(); i++) {
            int index = string.charAt(i)-'a';
            res |= 1<<index;
        }
        return res;
    }
```
