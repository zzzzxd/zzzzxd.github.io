### 题目

Leetcode原题：
https://leetcode.com/problems/minimum-cost-to-merge-stones/

### 题目解析

- 题目容易理解。比较难，枚举要求比较高。
- 两个优化细节
  - if ((n - 1) % (K - 1) > 0) { return -1; }  通过打表观察可以得出这个规律，其实比较难找，靠程序去猜规律。可以提前过滤
  - 递归中的mid++ 可以替换成 mid+=K-1 其实就是跳过了无效解。

### 代码

```java
public static int mergeStones(int[] stones, int k) {
        final int[] preSum = new int[stones.length];
        preSum[0] = stones[0];
        for (int i = 1; i < stones.length; i++) {
            preSum[i] = preSum[i-1] + stones[i];
        }
        int N = stones.length;
        final int[][][] dp = new int[N][N][k + 1];
        return process(stones,0,stones.length-1,1,k,preSum,dp);
    }
public static int process(int[] stones,int l,int r,int P,int k,int[] preSum,int[][][] dp){
        if (dp[l][r][P] !=0 ){ //0相当于不缓存了
            return dp[l][r][P];
        }
        if (l == r) { // arr[L..R]
            return P == 1 ? 0 : -1;
        }
        // L ... R 不只一个数
        if (P == 1) {
            int next = process(stones,l, r, k, k, preSum,dp);
            if (next == -1) {
                dp[l][r][P] = -1;
                return -1;
            } else {
                dp[l][r][P] = next + (l==0?preSum[r]:preSum[r] - preSum[l-1]);
                return dp[l][r][P];
            }
        }
        else{
            int minCost = Integer.MAX_VALUE;
            for (int mid=l;mid < r;mid++){ // 
                final int costl = process(stones, l, mid, 1, k, preSum,dp);
                final int costr = process(stones, mid + 1, r, P - 1, k, preSum,dp);
                if(costl==-1 || costr==-1)
                    continue;
                minCost = Math.min(costl+costr,minCost);
            }
            dp[l][r][P] = minCost==Integer.MAX_VALUE?-1:minCost;
            return dp[l][r][P];
        }

    }
```
