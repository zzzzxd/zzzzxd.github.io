### 题目

长度为N的数组arr，一定可以组成N^2个数字对
例如arr = [3,1,2]，数字对有(3,3) (3,1) (3,2) (1,3) (1,1) (1,2) (2,3) (2,1) (2,2)
也就是任意两个数都可以，而且自己和自己也算数字对
数字对怎么排序？
第一维数据从小到大；第一维数据一样的，第二维数组也从小到大
所以上面的数值对排序的结果为：(1,1)(1,2)(1,3)(2,1)(2,2)(2,3)(3,1)(3,2)(3,3)
给定一个数组arr，和整数k，返回第k小的数值对

### 题目解析

- 这个题挺不错。学会数数，注意有重复数数字的情况。

### 代码

方法一：O(N*log(N))

```java
public static int[] getKmin(int[] arr,int k){
        Arrays.sort(arr);
        int N = arr.length;
//      int d = k/N,f = k % N;
//      int pre = f == 0 ? arr[d - 1] : (d == 0 ? arr[0] : arr[d]);
        int pre = arr[(k-1)/N];
        int preNumSize = 0;
        int lessPreNumSize = 0;
        for (int i = 0;i < N && arr[i] <= pre;i++){
            if (arr[i] < pre){
                lessPreNumSize++;
            }else {
                preNumSize++;
            }
        }
        k -= lessPreNumSize * N;
        return new int[]{pre, arr[(k-1)/preNumSize]};
    }
```
