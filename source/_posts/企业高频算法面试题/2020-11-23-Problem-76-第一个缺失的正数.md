### 题目

​	Leetcode原题：

​	https://leetcode.com/problems/first-missing-positive/

### 题目解析

- 这类题目使用的一种我管它称作 “占位”的思想
  - 对于直接判定垃圾值，会占用有效空间，所以减小连续的最大值
  - 暂时无法判定是否为垃圾值，进行归位。
- 需要用L保证扫描过的是连续空间，否则无法判断当前已经连续到哪个值了
- 是从1开始的，用双指针可能会很简单

### 代码

```java
法一:
class Solution {
     public static  int firstMissingPositive(int[] nums) {
        int L = 0; //L,R为虚指,所谓的有效区得保证有效，关键是连续，也就是扫描过去的元素得是连续的才行,L位置必须是放L+1的值
        int R = nums.length-1; //R+1代表可连续增长的最大值
        while (L <= R){
           if(nums[L]==L+1){
               L++;
           }else if(nums[L] <= L||nums[L] > R+1 ||nums[L] == nums[nums[L]-1]){
               swap(nums,L,R);
               R--;
           }else{
               swap(nums,L,nums[L]-1);
           }
        }
        return L+1;
    }
    public static void swap(int[] num,int loc1,int loc2){
        int temp = num[loc1];
        num[loc1] = num[loc2];
        num[loc2] = temp;
    }
}
```
