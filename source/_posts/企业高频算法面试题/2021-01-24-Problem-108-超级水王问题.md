### 题目

超级水王问题

扩展1：摩尔投票

扩展2：给定一个正数K，返回所有出现次数>N/K的数

### 题目解析

- 水王题目算是比较经典的题目了，题目一和题目二实际上思路是一样的

### 代码

```java
public static void getMoreKNum(int[] arr,int k ){
        final HashMap<Integer, Integer> map = new HashMap<>();
        for(int i = 0;i < arr.length;i++){
            if(map.get(arr[i]) == null){
                if (map.size() < k-1){
                    map.put(arr[i],1);
                }else {
                    final Set<Integer> integers = map.keySet();
                    for (Integer integer : integers) {
                        if(map.get(integer)==1){
                            map.remove(integer);
                        }else{
                            map.put(integer,map.get(integer)-1);
                        }
                    }
                    if(map.size() < k-1){
                        map.put(arr[i],1);
                    }
                }
            }
        }
        final Set<Integer> integers = map.keySet();
        for (Integer integer : integers) {
            int count = 0;
            for (int i = 0; i < arr.length; i++) {
                if(arr[i] == integer) count++;
            }
            if(!(count > arr.length/k)) map.remove(integer);
            else System.out.println(integer);
        }

    }
    public static void main(String[] args) {
        getMoreKNum(new int[]{ 1, 2, 3, 1, 1, 2, 1 },4);
    }
```
