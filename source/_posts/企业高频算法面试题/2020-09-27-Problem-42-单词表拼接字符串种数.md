### 题目

​	假设所有字符都是小写字母  大字符串是str,arr是去重的单词表, 每个单词都不是空字符串且可以使用任意次.
使用arr中的单词有多少种拼接str的方式. 返回方法数.

### 题目解析

- 就是说长的字符串用单词来拼起来，看看有几种方式。首先不难想的是，对字符串进行判断，在index位置，判断单词表中哪个单词可以作为开头，切割之后进行递归。此处，需要单词表，枚举每个单词之后，需要判断是不是可以作为当前字符串开头，这又是和平均字符串长度有关。
- 看到这种单词表的形式，我们常常想到用Trie来进行优化，把单词表用前缀树组织起来，这样在判断是不是有单词作为前缀时，可以在树上走一个平均长度就能判断出来，省去了每次递归都遍历单词表的时间。

### 代码

```java
//方法一：DP  dp[i..length-1] 表示这个字符串由单词拼接的方法数
    public static int method1(String str,String words[]){
        final char[] chars = str.toCharArray();
        int N = str.length();
        final int[] dp = new int[N + 1];
        dp[N] = 1;
        for(int i = N-1;i >= 0;i--){
            dp[i] = 0;
            for(int j = 0;j < words.length;j++){
                final int t = str.substring(i).indexOf(words[j]);
                if(t == 0){
                    dp[i] += dp[i+words[j].length()];
                }
            }
        }
        return dp[0];
    }


    //方法二：Dp 可以看到套了两个循环，indexOf方法查子串 底层kmp O(n),几乎是三阶的
    //优化点在于，我当前i开始的串 在查找d单词时候，能匹配上的单词 一定是前缀关系
    public static int method2(String str,String words[]){

        final PrefixTree prefixTree = new PrefixTree();
        final char[] chars = str.toCharArray();
        int N = str.length();
        final int[] dp = new int[N + 1];
        dp[N] = 1;
        for (String word : words) {
            prefixTree.addWord(word);
        }
        //dp[i]表示 以i开始的字符串可以被words拼成的方法数
            final PrefixTree.Node root = prefixTree.root;
            for(int i = N-1;i >= 0;i--){
                dp[i] = 0;
                PrefixTree.Node cur = root;
                int j = i;
                while (j < chars.length){
                    PrefixTree.Node temp = cur.chars[chars[j]-'a'];
                    if(temp == null){
                        break;
                    }else{
                        if(temp.isWord){
                            dp[i] += dp[j+1];
                        }
                        cur = temp;
                    }
                    j++;
                }
            }
        return dp[0];
    }

    public static class PrefixTree{
        Node root;
        private class Node{
            boolean isWord;
            Node[] chars;
            String word;
            public Node() {
                this.chars = new Node[26];
            }
        }
        public PrefixTree() {
           root = new Node();
        }
        public void addWord(String word){
            Node cur = root;
            final char[] chars = word.toCharArray();
            for (int i = 0; i < chars.length; i++) {
                cur.chars[chars[i]-'a'] = cur.chars[chars[i]-'a'] == null ? new Node() : cur.chars[chars[i]-'a'];
                cur = cur.chars[chars[i]-'a'];
            }
            cur.isWord = true;
            cur.word = word;
        }

    }
```
