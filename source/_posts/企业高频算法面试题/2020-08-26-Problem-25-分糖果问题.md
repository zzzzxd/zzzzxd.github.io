### 题目

​	原问题：https://leetcode.com/problems/candy/

​	进阶问题：在原问题的基础上，增加一个原则：
​		相邻的孩子间如果分数一样，分的糖果数必须一样
​	返回至少需要分多少糖

### 题目解析

- 和原题基本一样通过两次遍历，先满足比左邻居的大，再遍历一次 满足比右边邻居的大。

### 代码

```java
 public int candy(int[] ratings) {

        if(ratings.length == 0) return 0;
        final int N = ratings.length;
        final int[] left = new int[N];
        final int[] right = new int[N];
        int res = 0;
        left[0] = 1;
        for (int i=1;i < N;i++){
            left[i] = ratings[i] > ratings[i-1]?left[i-1]+1:1;
        }
        right[N-1]=1;
        for(int j = N-2;j>=0;j--){
            right[j] = ratings[j] > ratings[j+1] ? right[j+1]+1:1;
        }
        for(int i = 0;i < N;i++){
            res += Math.max(left[i],right[i]);
        }
        return res;
    }
```
