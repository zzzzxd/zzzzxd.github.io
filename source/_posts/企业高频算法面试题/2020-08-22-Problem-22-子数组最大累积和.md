### 题目

​	返回一个数组中，子数组最大累加和

### 题目解析

- 这个题目就太常见了。

### 代码

```java
public static int max_subarr_sum(int[] arr){
        final int N = arr.length;

        int pre = arr[0];
        int res = pre;
        for (int i = 1; i < arr.length; i++) {
            pre = Math.max(arr[i],arr[i]+pre);
            res = Math.max(res,pre);
        }
        return res;
    }
```
