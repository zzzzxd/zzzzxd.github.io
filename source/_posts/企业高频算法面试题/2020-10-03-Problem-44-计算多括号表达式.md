### 题目

​	给定一个字符串str，str表示一个公式，公式里可能有整数、加减乘除符号和左右括号 返回公式的计算结果，难点在于括号可能嵌套很多层,如：

str="48*((70-65)-43)+8*1"，返回-1816。
str="3+1*4"，返回7。
str="3+(1*4)"，返回7。
【说明】 
1.可以认为给定的字符串一定是正确的公式，即不需要对str做公式有效性检查
2.如果是负数，就需要用括号括起来，比如“4*(-3)”但如果负数作为公式的开头或括号部分的开头，则可以没有括号，比如"-3*4"和"(-3*4)"都是合法的。 
3.不用考虑计算过程中会发生溢出的情况。

### 题目解析

- 常见的思路是利用栈结合操作符的优先级别进行计算，但递归本质上也是利用系统栈，我们利用递归来解决这种“优先级”的问题。

  

### 代码

```java
//通过递归计算括号优先级 1+10*3+4*5+7
    public static String method1(String str){
        final Stack<String> stack = new Stack<>();
        final char[] chars = str.toCharArray();
        return (String)process(stack,0,chars)[0];
    }
    public static  Object[] process(Stack<String> stack,int i,char[] chs){

        int start = i;
        if(chs[i] == '-'){
            stack.push("0");
            stack.push("-");
            start++;
        }
        while (start < chs.length){
            Double num = 0.0;
            boolean flag = false; //表示已经得出数字
            if(chs[start] == '(' ){
                final Object[] ans = process(new Stack<String>(), start + 1, chs);
                start = (Integer) ans[1];
                flag = true;
                num = Double.valueOf((String)ans[0]);
            }
            if(!flag){
                while (start < chs.length && chs[start] <= '9' && chs[start] >= '0'){
                    num = num *10 + (chs[start] - '0');
                    start++;
                }
            }
            Character op = null;
            if(start < chs.length){
                op = chs[start];
                start++;
            }

            //push 之前如果是乘除法 先算
            if(!stack.isEmpty()&&(stack.peek().equals("*") || stack.peek().equals("/"))){
                final String tmp_op = stack.pop();
                final String num_a = stack.pop();
                num = Double.valueOf(getCalculateRes(tmp_op,num_a,num.toString())) ;
            }
            stack.push(num.toString());
            if(op != null && op != ')')
                stack.push(op.toString());
            else
                break;
        }
        //计算栈中加减运算表达式的值
        String res = stack.pop();
        while (!stack.isEmpty()){
            if(isOption(stack.peek())){
                String option = stack.pop();
                String temp = stack.pop();
                res = getCalculateRes(option,temp,res);
            }
        }
        return new Object[]{res,start};

    }

    private static String getCalculateRes(String op, String a, String b) {
        Double res = null;
        switch (op){
            case "*":{
                res = Double.valueOf(a) * Double.valueOf(b);
                break;
            }
            case "/":{
                res = Double.valueOf(a) / Double.valueOf(b);
                break;
            }
            case "+":{
                res = Double.valueOf(a) + Double.valueOf(b);
                break;
            }
            case "-":{
                res = Double.valueOf(a) - Double.valueOf(b);
                break;
            }
            default:
                break;
        }
        return res.toString();
    }
    private static boolean isOption(String op){
        if(op.length() ==1 && (op.equals("+") || op.equals("-"))) return true;
        return false;
    }
```
