### 题目

​	字符串交错组成
​	https://leetcode.com/problems/interleaving-string/

### 题目解析

- 典型的dp行列模型，定义dp\[i]\[j\]表示s1的前i个和s2的前j个字符是否能组成s3的前i+j个字符

### 代码

```java
public static boolean isInterleave(String s1, String s2, String s3){
        final char[] arr1 = s1.toCharArray(),arr2 = s2.toCharArray(),arr3 = s3.toCharArray();
        int M = s1.length(),N = s2.length();
        if(M+N != s3.length()) return false;
        final boolean[][] dp = new boolean[M + 1][N + 1];
        dp[0][0] = true;
        for(int i = 1;i <= M;i++){
            dp[i][0] = arr1[i-1] == arr3[i-1] ? dp[i-1][0] : false;
        }
        for(int j = 1;j <= N;j++){
            dp[0][j] = arr2[j-1] == arr3[j-1] ? dp[0][j-1]:false;
        }
        for(int i = 1;i <= M;i++){
            for (int j = 1;j <= N;j++){
                boolean p1 = false,p2 = false;
                if(arr1[i-1] == arr3[i+j-1]){
                    p1 = dp[i-1][j];
                }
                if(arr2[j-1] == arr3[i+j-1]){
                    p2 = dp[i][j-1];
                }
                dp[i][j] = p1|p2;
            }
        }
        return dp[M][N];
    }
```
