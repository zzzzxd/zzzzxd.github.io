### 题目

​	返回一个数组中，选择的数字不能相邻的情况下，最大子序列累加和

### 题目解析

- 这种子序列，首选DP，不相邻只是一个限制条件。

### 代码

```java
public static int getNotNearMaxSub(int[] arr){
        final int N = arr.length;
        if(arr == null || N ==0) return 0;
        int next = arr[arr.length-2] > arr[arr.length-1] ? arr[arr.length-2] :arr[arr.length-1];
        int next_next = arr[arr.length-1];
        int res = arr[N-1];
        for(int i = N-3;i >= 0;i--){
            int temp_next = next;
            next = Math.max(Math.max(next,next_next + arr[i]),arr[i]); //求当前dp对应的值，提前更新迭代，正好存下来
            next_next = temp_next;
            res = Math.max(res,next);
        }
        return res;
    }
```
