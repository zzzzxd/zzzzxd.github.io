### 题目

​	已知一个消息流会不断地吐出整数 1~N，但不一定按照顺序依次吐出 如果上次打印的序号为i， 那么当i+1出现时 请打印 i+1 及其之后接收过的并且连续的所有数 直到1~N全部接收并打印完
请设计这种接收并打印的结构。

### 题目解析

- 经典题目。大思路就是构建头链表，构建尾链表，来一个元素就要去头链中 拽出来前面的元素，同时检查这个元素是否在尾链中，同理也要尝试拽尾链中的元素。

### 代码

```java
 public static class Node{
        int key;
        String info;
        Node next;

        public Node(int key, String info) {
            this.key = key;
            this.info = info;
        }
    }
    public static class Browse{
        static HashMap<Integer,Node> head;
        static HashMap<Integer,Node> tail;
        static int waitNo = 1;

        public Browse() {
            head = new HashMap<>();
            tail = new HashMap<>();
        }

        public static void receive(Node node){
            //检查前面有没有尾巴和它连
            if(tail.containsKey(node.key-1)){
                tail.get(node.key-1).next = node;
                tail.remove(node.key-1);
            }else{ //
                head.put(node.key,node);
            }
            //检查有没有下面的头和它相连
            if(head.containsKey(node.key+1)){
                node.next = head.get(node.key+1);
                head.remove(node.key+1);
            }else{
                tail.put(node.key,node);
            }

            if(waitNo == node.key){
                print(node);
            }
        }

        private static void print(Node node) {
            head.remove(node.key);
            String str = "" + node.info;
            waitNo++;
            while (node.next != null){
                str += node.next.info + " ";
                waitNo++;
                node = node.next;
            }
            tail.remove(node.key);
            System.out.println(str);
        }
    }

    public static void main(String[] args) {
        final Browse box = new Browse();

        box.receive(new Node(2,"B")); // - 2"
        System.out.println("这是2来到的时候");
        box.receive(new Node(1,"A")); // 1 2 -> print, trigger is 1
        System.out.println("这是1来到的时候");
        box.receive(new Node(4,"D")); // - 4
        box.receive(new Node(5,"E")); // - 4 5
        box.receive(new Node(7,"G")); // - 4 5 - 7
        box.receive(new Node(8,"H")); // - 4 5 - 7 8
        box.receive(new Node(6,"F")); // - 4 5 6 7 8
        box.receive(new Node(3,"C")); // 3 4 5 6 7 8 -> print, trigger is 3
        box.receive(new Node(9,"I")); // 9 -> print, trigger is 9
        box.receive(new Node(10,"J")); // 10 -> print, trigger is 10
        box.receive(new Node(12,"L")); // - 12
        box.receive(new Node(13,"M")); // - 12 13
        box.receive(new Node(11,"K")); // 11 12 13 -> print, trigger is 11
    }
```
