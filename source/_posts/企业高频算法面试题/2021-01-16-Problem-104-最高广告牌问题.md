### 题目

Leetcode原题：
https://leetcode.com/problems/tallest-billboard/

### 题目解析

- 题意就是挑选两个不同的子集，使得大小相同，求最大值。

- 难点在于，一个元素是如何确定出放进哪个集合的？当然不确定，所以要记录下所有可构成的两个集合，并且不断的缩小其差值并找到最大的那个。

- 代码一的这个bug 可让自己吃了一顿苦头。给自己提醒写代码时脑子要清醒。不然很容易想漏，代码一就是没有判断在构造新集合时 取较大的。再者。也可以直接使用一个map将旧值完全copy过来再操作。

  ```
  // cur 内部数据完全和之前map一样
  cur = new HashMap<>(dp); // 先进行拷贝 再枚举
  ```

### 代码

代码一（bug）:

```java
public static int tallestBillboard(int[] rods) {
        HashMap<Integer, Integer> map = new HashMap<>();
        map.put(0,0);
        for (int rod : rods) {
            final HashMap<Integer, Integer> new_map = new HashMap<>();
            //map 迭代过程中不能用put方法新增 必须用迭代器
            for (Integer key : map.keySet()) {
                final Integer minValue = map.get(key);
                int maxValue = minValue + key;
                int new_key = Math.abs((minValue + rod)-maxValue);
                new_map.put(new_key,Math.min(minValue + rod,maxValue));
                new_key = Math.abs((maxValue + rod)-minValue);
                new_map.put(new_key,Math.min(maxValue + rod,minValue));
            }
            for (Integer key: map.keySet()) {
                if(new_map.get(key) == null || (map.get(key) != null && map.get(key) > new_map.get(key))){
                    new_map.put(key,map.get(key));
                }
            }
            map = new_map;
        }
        return map.get(0)==null ? -1 : map.get(0);
    }
```

纠正:

```java
 public static int tallestBillboard(int[] rods) {
        HashMap<Integer, Integer> map = new HashMap<>();
        map.put(0,0);
        for (int rod : rods) {
            final HashMap<Integer, Integer> new_map = new HashMap<>();
            //map 迭代过程中不能用put方法新增 必须用迭代器
            for (Integer key : map.keySet()) {
                Integer minValue = map.get(key);
                int maxValue = minValue + key;
                int new_key = Math.abs((minValue + rod)-maxValue);
                new_map.put(new_key,Math.max(new_map.getOrDefault(new_key,0),Math.min(minValue + rod,maxValue)));
                new_key = Math.abs((maxValue + rod)-minValue);
                new_map.put(new_key,Math.max(new_map.getOrDefault(new_key,0),Math.min(maxValue + rod,minValue)));
            }
            for (Integer key: map.keySet()) {
                if(new_map.get(key) == null || (map.get(key) != null && map.get(key) > new_map.get(key))){
                    new_map.put(key,map.get(key));
                }
            }
            map = new_map;
        }
        return map.get(0)==null ? -1 : map.get(0);
    }
```
