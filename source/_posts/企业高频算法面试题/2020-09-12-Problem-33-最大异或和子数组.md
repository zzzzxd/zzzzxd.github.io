### 题目

​	数组中所有数都异或起来的结果，叫做异或和,给定一个数组arr，返回arr的最大子数组异或和	

### 题目解析

- 首先看到这种和啊，数组啊，应该想到”前缀和“ ，要有这种意识，总要有些经验上的积累。
- 先做出前缀的异或和数组，这样对于数组preSum\[i],和前置的哪个元素异或 得到最大值，最大值是多少？
- preSum元素放到前缀树上，利用数位 的贪心思想即可求解。

### 代码

```java
public static int MaxSubXOR(int[] arr){
        if(arr.length == 0 || arr == null) return 0;
        final PrefixTree prefixTree = new PrefixTree();
        prefixTree.addNum(0);
        int xor = 0,res = arr[0];
        for (int i = 0; i < arr.length; i++) {
            xor ^= arr[i];
            res = Math.max(prefixTree.getMaxXor(xor),res);
            prefixTree.addNum(xor);
        }
        return res;
    }
    public static class PrefixTree{
        private class Node{
            public HashMap<Integer,Node> map;
            public Node() {
                map = new HashMap<>();
            }
        }
        Node root = new Node();
        int size;
        public void addNum(int num){
            Node cur = root;
            for(int i = 31;i >= 0;i--){
                int temp = (num>>i) & 1;
                if(cur.map.get(temp) == null) cur.map.put(temp,new Node());
                cur = cur.map.get(temp);
            }
            size++;
        }
        public int getMaxXor(int num){
            Node cur = root;
            int res = num;
            for(int i = 31;i >=0;i--){
                int temp = (num>>i) & 1;
                int x;
                if(i == 31){
                    x = cur.map.get(temp)==null?temp^1:temp;
                }else
                    x = cur.map.get(temp^1)==null?temp:temp^1;
                if(x == 1)
                    res ^=(x<<i);
                cur = cur.map.get(x);
            }
            return res;
        }
    }
```
