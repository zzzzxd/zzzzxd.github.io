### 题目

​	给定一个数组arr，返回如果排序之后，相邻两数的最大差值 要求：时间复杂度O(N)

### 题目解析

- 这个题需要好好思考，其实没什么明显的知识点的提示，这时候就得想些 “特殊技巧”。这个题 N个数，可以分到N+1个桶中，把元素\[min,max\]，给把区间平均分配到桶中，间隔最大的两个元素 一定是夸桶！所以只需要在桶中维护最大，最小元素，最后挨个桶考察一下即可。

### 代码

```java
public static int method(int[] arr){
        int n = arr.length+1;
        int[][] bucket = new int[n][2]; //0列放最小值,1列放最大值
        for (int i = 0; i < bucket.length; i++) {
            bucket[i][0] = Integer.MAX_VALUE;
            bucket[i][1] = Integer.MIN_VALUE;
        }
        int min_num = Integer.MAX_VALUE,max_num = Integer.MIN_VALUE;
        for (int i = 0; i < arr.length; i++) {
            max_num = arr[i] > max_num ? arr[i] : max_num;
            min_num = arr[i] < min_num ? arr[i] : min_num;
        }
        double step_len = (double)(max_num - min_num)/n; //是一个range，然后平均到 每个桶，这么大个范围

        for (int i = 0;i < arr.length;i++){
            int buloc = (int)((arr[i]-min_num)/step_len);
            buloc = buloc < n ? buloc : buloc-1;
            //更新buloc这个桶的最值
            bucket[buloc][0] =  arr[i] < bucket[buloc][0] ? arr[i] : bucket[buloc][0];
            bucket[buloc][1] =  arr[i] > bucket[buloc][1] ? arr[i] : bucket[buloc][1];
        }
        int lastMax = bucket[0][1];
        int res = 0;
        for (int i = 1;i < n;i++){
            if(bucket[i][0] == Integer.MAX_VALUE){
                continue;
            }
            res = bucket[i][0] - lastMax > res ? bucket[i][0] - lastMax : res;
            lastMax = bucket[i][1];
        }
        return res;

    }
```
