### 题目

定义什么是可整合数组：
一个数组排完序之后，除了最左侧的数外，有arr[i] = arr[i-1]+1
则称这个数组为可整合数组
比如{5,1,2,4,3}、{6,2,3,1,5,4}都是可整合数组
返回arr中最长可整合子数组的长度

### 题目解析

- 没有思路时，要从最暴力的办法下手 先有路走，再讨论路的好坏。

- 两重循环枚举子数组，判断子数组是不是满足条件的。枚举子数组，枚举开始，结尾 。再做操作。这是三重循环。但实际 O(N^2)即可。

- 学会重新定义题目条件。

  - 可整合数组 判断是不是等差数列，但是不是简单求和就可以判断的！

    ```java
    if((maxn + minn)*(R-L+1) == sum*2)
        ans = Math.max(ans,R-L+1);
    ```

  - 正确的定义确实是，1.无重复 2.最大值与最小值差值等于个数减一

### 代码

```java
public static int getLIL(int[] arr) {
        int ans = 1;
        final HashSet<Integer> set = new HashSet<>();
        for (int L = 0;L < arr.length;L++){
            int maxn = arr[L];
            int minn = arr[L];
            set.clear();
            set.add(arr[L]);
            for (int R=L+1;R < arr.length;R++){
                if (set.contains(R))
                    break;
                maxn = Math.max(maxn,arr[R]);
                minn = Math.min(minn,arr[R]);
                if(maxn - minn == R-L)
                    ans = Math.max(ans,R-L+1);
            }
        }
        return ans ;
    }
```
