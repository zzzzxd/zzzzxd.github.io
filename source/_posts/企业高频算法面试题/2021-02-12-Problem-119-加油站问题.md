### 题目解析

- 加油站问题比较经典，这里还是介绍比较通俗的做法

### 代码

```java
class Solution {
  public static int canCompleteCircuit(int[] gas, int[] cost) {
        int N = cost.length;
        if (N == 1) return gas[0]-cost[0] >= 0?0:-1;

        for (int i = 0; i < N; i++) {
            cost[i] = gas[i] - cost[i];
        }
        int p1 = 0;
        while (p1 < N){
            while (p1 < N &&cost[p1] < 0){
                p1++;
            }
            if(p1 < N){
                int p2 = p1,preSum = cost[p1];
                //窗口增长
                while ((p2-p1+N)%N+1<N && preSum >= 0){
                    p2 = (p2+1)%N;
                    preSum += cost[p2];
                }
                if(preSum >= 0 ) return p1;
                else {
                    if(p2 > p1) p1 = p2;
                    else return -1;
                }
                p1++;
            }
        }
        return -1;
    }
}
```
