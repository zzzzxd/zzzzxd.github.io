### 题目

Leetcode原题：

https://leetcode-cn.com/problems/boolean-evaluation-lcci/

### 题目解析

- 这个题也是很类似的思想，比如讲，求最大子数组和？不管哪个子数组和最大，子数组一定是以某个元素结尾，所以我们找以当前元素为结尾的最大子数组，最终找到答案。此题求的是加括号的方法数，首先每个运算符号一定都会生效。通过加括号使得答案最终为1，那么一定是最后那个运算符起了决定作用得到1。所以我们遍历当前符号，看它左边有多少种为0位1的方案，右面有多少种，分别递归得到，得到之后，我们通过运算算出 当前符号最后生效时的方案数，全部累加即可得到。

### 代码

```java
class Solution {
    private static class Result{
        int zeroCount;
        int OneCount;

        public Result(int zeroCount, int OneCount) {
            this.zeroCount = zeroCount;
            this.OneCount = OneCount;
        }
    }
    public static  int countEval(String s, int result) {
        if(s == null || s.length() == 0) return 0;
        HashMap<String,String> map = new HashMap();
        Result[][] dp = new Result[s.length()][s.length()];
        final Result res = process(s.toCharArray(), 0, s.length() - 1,dp);
        return result == 0?res.zeroCount : res.OneCount;

    }
    //【s，e】一定是两个字符
    public static Result process(char[] arr,int s,int e,Result[][] dp){
        if(s == e){
            int cnt = arr[s]-'0';
            return new Result(cnt^1,cnt);
        }
        if(dp[s][e]!=null){
            return dp[s][e];
        }
        int zeroCount = 0;
        int oneCount = 0;
        for (int i = s+1;i < e;i +=2){
            //以i字符划分
            final Result left = process(arr, s, i - 1,dp);
            final Result right = process(arr, i + 1, e,dp);

            switch (arr[i]){
                case '&':{
                    oneCount += left.OneCount * right.OneCount;
                    zeroCount += left.zeroCount * right.OneCount + left.zeroCount * right.zeroCount + left.OneCount * right.zeroCount;
                    break;
                }
                case '|':{
                    oneCount += left.OneCount* right.OneCount + left.zeroCount * right.OneCount + left.OneCount * right.zeroCount;
                    zeroCount += left.zeroCount * right.zeroCount;
                    break;
                }
                case '^':{
                    oneCount += left.OneCount * right.zeroCount + left.zeroCount * right.OneCount;
                    zeroCount += left.zeroCount * right.zeroCount + left.OneCount * right.OneCount;
                    break;
                }
                default:break;
            }
        }
        dp[s][e] = new Result(zeroCount,oneCount);
        return dp[s][e];

    }
}
```
