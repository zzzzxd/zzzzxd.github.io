### 最大公约数与最小公倍数 

#### 最大公约数

正整数a,b的最大公约数是指a与b的所有公约数中最大的那个公约数。求解最大公约数常用的是欧几里得算法(即辗转相除法),假设 a,b 均为正整数，则`gcd=(a,b) = gcd(b,a%b)`

代码如下:

```java
 int gcd(int a,int b){
        return b==0 ? a:gcd(b,a%b);
 }
```

#### 最小公倍数

正整数a,b的最小公倍数是指a与b的所有公倍数中最小的那个公倍数。当得到a和b的最大公约数d之后,可以马上得到a和b的最小公倍数是`ab/d`,一般用lcm(a,b)来表示a和b的最小公倍数。由于`a*b`在实际运算时可能溢出 ，所以更恰当的写法是`a/d*b`

```java
 int lcm(int a,int b){
        return a/gcd(a,b)*b;
}
```

### 素数筛选

素数又称质数，是指除了1和它自身，不能被其它数整除的数字。比较简单的做法是利用定义去挨个判断即可。但通常我们需要更快的做法去打表，以便后续更快的计算。这里我们常用是"埃氏筛法"

```java
    boolean[] primer_tb(int maxn){
        boolean[] not_primer = new boolean[maxn]; //true代表不是质数
        for (int i = 2;i < maxn;i++)
            if (!not_primer[i])
                for (int j = 2*i;j < maxn;j += i)
                    //筛掉所有i的倍数
                    not_primer[j] = true;
        return not_primer;
    }
```

### 质因子分解

所谓的质因子分解是指将一个正整数n写成一个或多个质数乘积的形式，例如 `180 = 2 * 2 * 3 * 3 * 5` 也即 

180 = 2^2^ * 3 ^2^ *5^1^  ,显然 数最终都是归到不同质数的乘积，所以我们需要先得到素数表 。

此外，我们需要理解一个结论 ,如果正整数n存在1和本身之外的因子，那么一定是在 sqrt(n)的左右成对出现，这是显然的。我们把这个结论应用到质因子上面得到一个强化的结论 ：<font color = red>对于一个正整数n来说，如果它存在[2,n]范围内的质因子，要么这些质因子全部小于等于sqrt(n),要么只存在一个大于sqrt(n)的质因子，而其余质因子全部小于等于 sqrt(n)。</font>

有这个结论，接下来写代码就好办了。

```java
 private class factor{
        int x; //质数
        int cnt; //质数对应的个数
    }
    public factor[] handle_primer(int n,int[] primer_tb,int p_num){ //primer_tb是收集好的质数数组,p_num是对应的个数。
        factor[] factors = new factor[10];
        int num = 0,sqr = (int)Math.sqrt(n);
        for (int i = 0;i < p_num && primer_tb[i] <= sqr;i++){
            if(n%primer_tb[i] == 0){
                factors[num].x = primer_tb[i];  //记录该质因子
                factors[num].cnt = 0;
                while (n % primer_tb[i] == 0){ //计算质因子primer_tb[i]的个数
                    factors[num].cnt++;
                    n/=primer_tb[i];
                }
            }
            num++;
        }
        if(n != 1){
            factors[num].x = n;
            factors[num].cnt = 1;
        }
        return factors;
    }
```
